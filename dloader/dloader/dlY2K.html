Year 2000 Compliance – AS/400 Report Downloader and AS/400 Smart File Downloader
August 21, 1998
This document provides information regarding year 2000 compliance for Evergreen Interactive Systems’ Products - AS/400 Report Downloader and AS/400 Smart File Transfer for TCP/IP.

AS/400 Report Downloader

Evergreen Interactive’s AS/400 Report Downloader utility allows an authorized PC user to view AS/400 spooled files and transfer their contents to a PC text file, PC printer, MS Word or MS Excel.

The AS/400 Report Downloader makes use of an IBM API for determining the date associated with a given spool file. This information includes a 4-digit year designation in century-year format. The Downloader makes no use of the spool file date, however, beyond displaying it for the user.

AS/400 Smart File Downloader

The AS/400 Smart File Downloader allows an authorized PC user to transfer all types of AS/400 data to a PC text file, PC printer, MS Word or MS Excel, Lotus 1-2-3 or WordPro, or to Email over a native TCP/IP network.  The AS/400 Smart File Downloader utility faithfully recognizes and transfers all standard AS/400 date formats. Dates are not used by the utility itself.
